Handbuch

Projekt:Mastermind Spielerklärung: Mastermind ist ein Spiel, in welchem ein zufallsgenerierter Code erraten werden muss. Diese Code ist eine Reihenfolge von verschiedenen Farben. Wenn man die richtige Farbe an der richtigen Position hat gibt das Spiel in dem Kontrollfeld die Farbe schwarz. Achtung! Nur weil das Kontrollfeld an der ersten Stelle schwarz ist, heißt es nicht, dass auch die erste Position mit der Farbe stimmt. Es heißt lediglich, dass eine Farbe an der richtigen Position ist. Hat man die richtige Farbe, jedoch nicht die Richtige Position, so gibt das Kontrollfeld die Farbe Weiß. Hierdurch weiß man, dass die Farbe in dem von dem Computer generierten Muster vorkommt, jedoch die Position nicht stimmt. Sollte weder die Farbe schwarz, noch weiß auftauchen muss man sich einen neuen Code von Farben überlegen.

1.Spielanleitung

Der Computer gibt ein Zufalls generiertes Muster wieder. Sie beginnen damit, in der untersten Zeile, eine zufällige Reihenfolge an Farben festzulegen. Mit dem "Check"-button schauen Sie, ob Sie die richtigen Farben und Position haben oder ob Sie von neuem raten müssen. Ihr eingegebene Reihenfolge wird in die nächst obere Zeile kopiert, sodass Sie leichter danach die Farben ändern können.

2.Levelbeschreibung

  • Bei Level 1 haben Sie 4 Positionen von Farben und 8 Versuche, die gewünschte Reihenfolge von Farben zu erraten.
  • Bei Level 2 haben Sie 5 Positionen von Farben und 9 Versuche, die gewünschte Reihenfolge von Farben zu erraten.
  • Bei Level 3 haben Sie 6 Positionen von Farben und 10 Versuche, die gewünschte Reihenfolge von Farben zu erraten.

    • 3.Buttonerklärung

  • Der Button: "Help" erklärt noch einmal das Spiel in Kurzfassung.
  • Der Button: "Highscore" gibt den Highscore wieder. Hierfür muss man aber erst in Java das Programm Server ausführen.
  • Der Button: "Check" sorgt dafür, dass das Programm den von Ihnen eingegeben Code überprüft und das Kontrollfeld wird ausgefüllt.
  • Der Button: "Zurück" sorgt dafür, dass man wieder zur Levelauswahl kommt.
  • Der Button: "Hauptmenü" bringt einen wieder in das Hauptmenü um zum Beispiel das Spiel von neuem zu beginnen oder den Highscore anzuschauen.

    • 4.Installation

    Um das Spiel spielen zu können brauchen Sie ein "Java runtime environment". Anschließend müssen Sie die 2 Programme: "Ausführen", "SpeicherServer" in Java öffnen und dann zuerst das Programm "SpeicherServer" ausführen und dann anschließend das Programm "Ausführen" ausführen.
